ğŸ§  Model Logic
The application expects specific inputs processed in the same way as the training data:

Log Transformations: Applied to total_rooms, total_bedrooms, population, and households to normalize skewed distributions.

Feature Engineering: Ratios like bedroom_ratio (bedrooms/rooms) and household_rooms (rooms/households) are calculated dynamically.

Encoding: The ocean_proximity feature is one-hot encoded to match the training set structure.
